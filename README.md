# 章程技术主页

这是章程的个人技术主页，展示了美团后端开发工程师的专业能力和技术成就。

## 🎯 网站特色

### 视觉设计
- **深色科技主题** - 现代化的深色配色方案，保护眼睛的同时展现科技感
- **渐变色彩** - 使用青色和粉色渐变，营造未来科技氛围
- **粒子动画** - 动态粒子背景效果，增加视觉层次
- **响应式设计** - 完美适配桌面、平板和手机设备

### 交互功能
- **平滑滚动导航** - 点击导航自动平滑滚动到对应区域
- **滚动动画** - 元素进入视口时触发动画效果
- **鼠标跟随** - 自定义鼠标指针效果
- **打字机效果** - 英雄区域标题的动态打字效果
- **表单验证** - 联系表单的实时验证和错误提示

### 内容展示
- **教育背景** - 详细展示华南理工大学材料科学与工程本科和香港中文大学信息工程硕士经历
- **工作经验** - 时间线式展示美团、腾讯、阿里巴巴等公司的工作经历
- **技术技能** - 分门别类展示后端、数据库、云服务等技术栈
- **项目展示** - 突出美团订单系统、腾讯云存储等代表性项目
- **研究成果** - 展示分布式系统和后端架构相关的论文和技术报告

## 📁 文件结构

```
个人网站用/
├── index.html      # 主页面HTML结构
├── styles.css      # 现代化CSS样式
├── script.js       # JavaScript交互功能

```

## 🚀 技术栈

- **HTML5** - 语义化标签结构
- **CSS3** - 现代化样式特性
  - CSS Grid & Flexbox 布局
  - CSS 动画和变换
  - CSS 变量和自定义属性
  - 响应式媒体查询
- **Vanilla JavaScript** - 原生JavaScript实现所有交互
  - ES6+ 现代语法
  - Intersection Observer API
  - 事件委托和优化

## 🎨 设计理念

1. **专业性** - 突出AI研究员的技术能力和学术背景
2. **现代感** - 使用最新的Web技术和设计趋势
3. **用户体验** - 流畅的动画和交互，提升用户参与度
4. **可访问性** - 考虑屏幕阅读器和键盘导航
5. **性能优化** - 轻量级实现，确保快速加载

## 🔧 本地开发

1. 克隆项目到本地
2. 使用浏览器直接打开 `index.html` 文件
3. 或者使用本地服务器运行：

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# 然后访问 http://localhost:8000
```
